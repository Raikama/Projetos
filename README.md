Este projeto tem como objetivo analisar a distribuição de vendas entre diferentes departamentos e modos de envio utilizando técnicas de analise de dados e visualização com python.
A proposta central é transformar dados brutos em insights visuais claros.

objetivos

Identificar as maiores vendas localizada na região Centro-Oeste em 2017 e 2018
Comparar valores em milhares de reais em cada Estado.
Criar visualizações em formato de gráficos 
Aprender sobre os tipos de visualizações disponivel nas bibliotecas 
Analisar a participação percentual nos departamento e modo de envio
Comparar valores absolutos e relativos

Tecnologias utilizadas:

Google Colab 
Python
Matplotlib
Pandas
Seaborn
